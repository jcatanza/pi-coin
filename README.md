# pi-coin
Fun project based on a new result "Estimating $$\pi$$  with a Coin", by Jim Propp, https://arxiv.org/abs/2602.14487, Feb 26, 2026

In this paper, Jim Propp shows that you can estimate the value of $$\pi$$ from repeated experiments with a coin, by counting lengths of sequences of coin flips which end when the number of heads eexceeds the number of tails by one. Specifically, $$E\left[\frac{H_\tau}{\tau}\right] = \frac{\pi}{4}$$
